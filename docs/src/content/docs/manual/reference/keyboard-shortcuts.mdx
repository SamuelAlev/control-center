---
title: Keyboard shortcuts
description: Keyboard shortcuts available in Control Center.
---

Control Center is keyboard-first. Every action is reachable without a mouse.

## Global shortcuts

| Shortcut | Action |
|---|---|
| `Cmd/Ctrl + K` | Open command palette |
| `Cmd/Ctrl + 1` | Navigate to dashboard |
| `Cmd/Ctrl + 2` | Navigate to tickets |
| `Cmd/Ctrl + 3` | Navigate to pull requests |
| `Cmd/Ctrl + 4` | Navigate to pipelines |
| `Cmd/Ctrl + 5` | Navigate to agents |
| `Cmd/Ctrl + 6` | Navigate to analytics |
| `Cmd/Ctrl + 7` | Navigate to memory |
| `Cmd/Ctrl + 8` | Navigate to newsfeed |
| `Cmd/Ctrl + ,` | Open settings |
| `Cmd/Ctrl + Shift + W` | Toggle workspace switcher |
| `Cmd/Ctrl + Shift + T` | Toggle light/dark theme |
| `Cmd/Ctrl + Shift + Alt + F` | Toggle focus mode |
| `Cmd/Ctrl + Shift + [` | Switch to previous workspace |
| `Cmd/Ctrl + Shift + ]` | Switch to next workspace |
| `Cmd/Ctrl + Alt + 1ŌĆō9` | Switch to workspace 1ŌĆō9 |

## PR list

| Shortcut | Action |
|---|---|
| `J` / `K` | Next / previous pull request |
| `Enter` | Open selected pull request |
| `X` | Toggle selection of focused pull request |
| `E` | Merge the focused pull request |
| `Space` | Peek (expand/collapse) the focused pull request |
| `R` | Refresh pull request list |

## PR diff viewer

| Shortcut | Action |
|---|---|
| `J` / `K` | Next / previous file |
| `V` | Mark file as viewed / unviewed |
| `C` | Collapse / expand file |
| `Cmd/Ctrl + F` | Find in diff |
| `Enter` | Next search match |
| `Shift + Enter` | Previous search match |
| `Escape` | Close search / clear selection |
| `Cmd/Ctrl + C` | Copy diff selection |
| `1` / `2` | Switch to conversation / files changed tab |

## Messaging

| Shortcut | Action |
|---|---|
| `Enter` | Send message |
| `Shift + Enter` | New line in message |
| `@` | Start @-mention |
| `Escape` | Cancel message / dismiss popup |
| `Cmd/Ctrl + N` | New direct message |
| `Cmd/Ctrl + Shift + N` | New group channel |
| `J` / `K` | Next / previous channel |
| `Cmd/Ctrl + Backspace` | Delete selected channel |

## Workspaces

| Shortcut | Action |
|---|---|
| `Cmd/Ctrl + N` | New workspace |
| `Enter` | Open selected workspace |
| `Cmd/Ctrl + Backspace` | Delete selected workspace |

## Newsfeed

| Shortcut | Action |
|---|---|
| `J` / `K` | Next / previous article |
| `Enter` | Open selected article |
| `S` | Bookmark / unbookmark article |
| `R` | Refresh feeds |

## Navigation

| Shortcut | Action |
|---|---|
| `Tab` | Move focus to next element |
| `Shift + Tab` | Move focus to previous element |

## Accessibility

- All shortcuts have visible focus indicators (2px focus rings)
- Focus order follows the visual layout
- Screen reader support via semantic labels
- Reduced motion alternatives for all animations
