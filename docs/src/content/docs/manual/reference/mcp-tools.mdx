Control Center exposes 70+ typed tools via its MCP server. This reference catalogs them by domain.

## Common parameters

### `workspace_id`

**Required** on all tools that touch workspace-scoped data. Omit only for genuinely global tools (`list_workspaces`, `create_workspace`).

## Tool gating

Tools are gated by conversation mode via `ConversationModeToolGuard`:

- **Chat mode**: full tool access
- **Plan mode**: read-only tools (no file writes)
- **Review mode**: review-specific tools only

## Destructive action confirmation

Some tools require explicit user approval:

- `kill_agent`
- `merge_pull_request`
- `fire_agent`
- Destructive bash commands

The tool returns an `ApprovalPayload`; the client must confirm before execution proceeds.
