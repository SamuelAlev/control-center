This guide shows you how to use focus mode for distraction-free PR review.

## What is focus mode?

Focus mode is an ephemeral, minimalist UI for PR review. It strips away the sidebar, navigation, and all non-essential UI to give you a clean, full-screen diff viewer.

Focus mode is designed for deep review sessions where you need to concentrate on the code without interruptions.

## Enter focus mode

1. Open any PR
2. Click the focus mode toggle in the PR header (or use the keyboard shortcut)
3. The sidebar and title bar collapse, leaving only the diff viewer

## In focus mode

- **Full-width diff viewer** — the diff takes up the entire window
- **Keyboard navigation** — navigate between files and changes without a mouse
- **Inline comments** — leave comments directly on the diff
- **File list** — a collapsible file tree for quick jumps

## Exit focus mode

Press `Escape` or click the exit button to return to the normal shell layout.

Focus mode is ephemeral — it's a UI toggle, not a persistent state. Your viewed-file progress and draft comments are preserved when you exit and re-enter.

## When to use focus mode

- Large PRs with many files to review
- Security-sensitive code that requires deep attention
- When you want to avoid distraction from notifications and sidebar activity
